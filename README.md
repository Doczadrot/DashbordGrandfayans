# Quality Dashboard — Аналитика Качества и Продаж

Интеллектуальный дашборд для мониторинга качества продукции и анализа продаж на маркетплейсах (Ozon, Яндекс.Маркет, Wildberries). Проект сочетает в себе строгую iOS-стилистику с мощным функционалом анализа данных.

## 🚀 Основные возможности

### 📊 Аналитика Качества
- **Многоуровневый анализ причин**: Детализация дефектов от общих категорий до конкретных описаний.
- **Сравнение поставщиков**: Динамическое сравнение ТОП-5 или полный анализ всех поставщиков по клику.
- **Интеграция маркетплейсов**: Отдельный режим анализа "Бой Озон/Яндекс" для специфических дефектов логистики.
- **Умная нормализация**: Автоматическое объединение похожих причин дефектов с подтверждением пользователя.

### 💰 Аналитика Продаж
- **Автоматический парсинг**: Поддержка файлов XLSX/CSV/TXT с данными о продажах.
- **Алгоритм МП**: Умная обработка специфических файлов маркетплейсов (правило "1 строка = 1 штука", очистка сумм от пробелов и запятых).
- **Динамика по месяцам**: Визуализация объемов продаж в разрезе месяцев и площадок.

### 🎨 Дизайн и UX (iOS Style)
- **Glassmorphism**: Прозрачные карточки с эффектом размытия фона (`backdrop-filter: blur`).
- **Интерактивность**: Плавные анимации переходов (Framer Motion).
- **Сетка**: Гибкая настройка дашборда с помощью `react-grid-layout`.
- **Типографика**: Крупные заголовки (SF Pro) и четкая цветовая кодировка статусов.

## 🛠 Технологический стек

- **Frontend**: React 18, TypeScript, Vite.
- **State Management**: Zustand.
- **Visualization**: Chart.js 4, React-Chartjs-2.
- **Styling**: Tailwind CSS, Lucide React (иконки).
- **Animations**: Framer Motion.
- **Data Parsing**: XLSX.js.

## 📂 Структура данных

Система ожидает файлы со следующими ключевыми полями:
- **Номенклатура**: Наименование товара.
- **Поставщик / Контрагент**: Источник товара или площадка (автоматический маппинг "ИНТЕРНЕТ РЕШЕНИЯ" -> Озон).
- **Описание дефекта**: Текстовое описание проблемы для глубокого анализа.
- **Дата**: Автоматическое определение месяца продажи из форматов `ДД.ММ.ГГГГ`.

## 📦 Развертывание

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите в режиме разработки:
   ```bash
   npm run dev
   ```
3. Сборка проекта:
   ```bash
   npm run build
   ```
